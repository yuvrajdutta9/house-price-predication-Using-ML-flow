# 🏡 House Price Prediction using Machine Learning

## 📌 Objective
Develop a machine learning model to predict house prices using a real-world dataset from Kaggle.  
This project demonstrates **data preprocessing, visualization, model building, evaluation, and reporting skills**.

---

## 📂 Dataset
- Source: [House Prices Dataset (Kaggle)](https://www.kaggle.com/datasets/shree1992/housedata)

---

## ⚙️ Software Requirements
You can use either:
- **Option 1:** Jupyter Notebook (Recommended)  
- **Option 2:** Visual Studio Code (VS Code) with Jupyter Extension  

> Final submission must be in `.ipynb` format.

---

## 🛠️ Project Workflow
1. **Import Libraries** → NumPy, Pandas, Matplotlib, Seaborn, Scikit-learn  
2. **Load Dataset** → Inspect structure and metadata  
3. **Data Cleaning** → Handle missing values, duplicates, and data types  
4. **Exploratory Data Analysis (EDA)** → Identify trends and correlations  
5. **Visualizations** → At least six meaningful plots (heatmaps, scatterplots, distributions, etc.)  
6. **Feature Engineering** → Encoding categorical variables, scaling features  
7. **Train-Test Split** → Prepare data for model training  
8. **Model Training** → Implement at least three regression models:
   - Linear Regression  
   - Decision Tree Regressor  
   - Random Forest Regressor  
9. **Model Evaluation** → Metrics: MAE, MSE, RMSE, R² Score  
10. **Model Comparison** → Select and justify the best-performing model
